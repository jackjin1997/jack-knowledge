# 复盘模板

决策后的反思记录，提取智慧和经验。

## 为什么复盘

### 1. 学习和成长
从经验中学习，无论结果如何

### 2. 提升决策能力
看到自己的思维模式和盲点

### 3. 建立自信
看到成长轨迹，增强信心

### 4. 调整方向
及时发现问题，调整行动

## 复盘时机

### 短期复盘（1周-1个月）
- 关注：初步结果，执行情况
- 目的：及时调整

### 中期复盘（3-6个月）
- 关注：中期影响，深层次变化
- 目的：评估方向，提取经验

### 长期复盘（1年+）
- 关注：长期影响，人生轨迹
- 目的：深刻智慧，人生理解

## 复盘原则

### 1. 诚实，不美化

- 记录真实的结果
- 承认错误和失误
- 不为自己辩护
- 也不过度苛责

### 2. 学习，不评判

- 关注学习，不是对错
- 每个结果都是老师
- 失败和成功都有价值

### 3. 具体，不抽象

```
❌ "我学到要更谨慎"
✅ "我学到在做X类决策时，需要先验证Y假设，
   因为上次我忽略了这个导致Z问题"
```

### 4. 前瞻，不沉溺

- 复盘是为了未来
- 不是为了后悔
- 提取经验，继续前进

## 复盘频率建议

### 重大决策
- 1周、1月、3月、6月、1年各复盘一次
- 看到完整的影响展开

### 中等决策
- 1月、3月各复盘一次
- 足够提取主要经验

### 小决策
- 3月或6月复盘一次
- 或与其他决策一起批量复盘

### 季度集中复盘
每个季度，回顾该季度的所有决策：
- 发现模式
- 看到成长轨迹
- 更新价值观认知
