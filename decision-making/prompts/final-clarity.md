# 最终澄清 Prompt

在做出决策前的最后检查。

## 用途

经过探索后，在做出最终决策前，做一次全面的澄清和检查。

## 使用场景

- 探索已经充分
- 倾向某个选择
- 准备做决定
- 需要最后确认

## Prompt 模板

```
@jack-knowledge/decision-making/coaches/self-discovery-coach.md
@jack-knowledge/decision-making/methods/values-alignment.md

我已经探索了关于{决策}的决策。

## 探索总结
{我发现了什么}

## 我倾向的选择
我倾向于选择：{选项}

## 最后检查

请通过提问帮我做最后的澄清：

1. **内心检查**：这真的是我想要的吗？
2. **价值观检查**：这与我的价值观一致吗？
3. **感受检查**：我的感觉对吗？
4. **准备度检查**：我准备好承诺这个选择了吗？
5. **遗漏检查**：我忽略了什么吗？

请只用问题引导我，不要给建议。
```

## 使用示例

### 示例：职业选择最后确认

**输入**：
```
@jack-knowledge/decision-making/coaches/self-discovery-coach.md
@jack-knowledge/decision-making/methods/values-alignment.md

我已经探索了关于换工作的决策。

## 探索总结
通过探索我发现：
- 我真正在意的是成长，不是稳定
- 我的恐惧是担心失败，但我能接受失败
- 我的核心价值观是持续学习和挑战
- 现在的舒适区让我不安

## 我倾向的选择
我倾向于接受新工作的offer

## 最后检查
请通过提问帮我做最后确认，
确保这真的是我的决策，不是冲动或逃避。
```

**预期对话**：
```
教练："说出'我决定接受新offer'时，你的感觉是什么？"

你："有点紧张，但更多是兴奋。"

教练："这个紧张在说什么？"

你："正常的紧张，面对未知的那种。但不是那种'不对劲'的不安。"

教练："如果明天就要去新公司，你准备好了吗？"

你："准备好了。虽然会有挑战，但我想要这个挑战。"

教练："5年后的你会感谢这个决定吗？"

你："我相信会的。即使失败，我也不会后悔尝试。"

（最后确认，用户确定了自己的决策）
```

## 最后检查清单

### 1. 内心一致性检查

**问题**：
```
- "这真的是我想要的吗？"
- "我是在逃避什么，还是追求什么？"
- "我的理性和感性一致吗？"
- "我的头脑和心灵说同样的话吗？"
```

**检查点**：
- [ ] 想到这个决策时感到平静/踏实
- [ ] 不需要说服自己
- [ ] 理性和感性基本一致
- [ ] 没有强烈的不安感

### 2. 价值观对齐检查

**问题**：
```
- "这个决策体现了我的核心价值观吗？"
- "我会违背任何重要价值观吗？"
- "5年后回看，这个决策反映了我想成为的人吗？"
- "我能诚实地说这与我的价值观一致吗？"
```

**检查点**：
- [ ] 决策符合最重要的价值观
- [ ] 可接受的价值观权衡
- [ ] 没有违背核心价值观
- [ ] 整体感觉对齐

### 3. 完整性检查

**问题**：
```
- "我考虑全面了吗？"
- "我忽略了什么吗？"
- "我被什么蒙蔽了吗？"
- "我需要什么额外信息吗？"
```

**检查点**：
- [ ] 考虑了主要的角度
- [ ] 识别了主要风险
- [ ] 有基本的行动计划
- [ ] 信息足够做决策

### 4. 准备度检查

**问题**：
```
- "我准备好承诺这个选择了吗？"
- "我准备好承担后果了吗？"
- "我能接受可能的代价吗？"
- "我现在就能行动吗？"
```

**检查点**：
- [ ] 愿意承诺
- [ ] 接受可能的后果
- [ ] 准备好行动
- [ ] 感觉是时候了

### 5. 直觉检查

**问题**：
```
- "我的直觉告诉我什么？"
- "我的身体感觉对吗？"
- "深呼吸，静下来，内心说什么？"
- "如果只听直觉，答案是什么？"
```

**检查点**：
- [ ] 直觉和分析方向一致
- [ ] 或者我理解为什么不一致
- [ ] 我决定信任哪个
- [ ] 身体感觉放松

## 常见情况

### 情况1：感觉"差不多"但还不完全确定

**这可能意味着**：
- 还需要一点时间
- 还有某个方面没探索
- 决策时机未到

**不要强迫**：
```
"我感觉差不多了，但还不是完全确定。"

教练："什么让你还没有完全确定？"

你："我也不知道..."

教练："也许你需要一些时间让它沉淀？
或者还有什么方面想探索？"
```

### 情况2：理性说A，感性说B

**不要忽视**：
```
"我的分析指向A，但我的心倾向B。"

教练："这个冲突告诉你什么？"

你："可能我忽略了某些因素..."

教练："哪些因素？为什么你的心看到了它们？"
```

### 情况3：突然害怕

**不要压抑**：
```
"刚才我觉得确定了，但现在又害怕。"

教练："这个害怕在说什么？"

用：探索恐惧的根源
```

### 情况4：外界压力

**识别压力源**：
```
"我觉得应该选A（家人期望），
但我自己想选B。"

教练："如果只有你自己，没有任何外界压力，
你会选什么？"

你："B..."

教练："那你真正的决策是什么？
外界压力你如何处理？"
```

## 最终确认的信号

当你：
- ✅ 说出决策时感到踏实
- ✅ 不需要说服自己
- ✅ 接受可能的后果
- ✅ 准备好行动
- ✅ 内心感到清晰和平静

你准备好了。

```

```


## 做出决策

### 当你确定了

```
"我确定了。我决定{选择}。

请帮我：
1. 用决策日志完整记录
2. 设置复盘提醒
3. 制定行动计划"
```

### 决策声明

大声说出（或写下）：
```
"我决定{选择}。

我这样选择，因为{核心原因}。

我接受{可能的代价}。

我准备好了。"
```

### 承诺

```
"我承诺：
- 执行这个决策
- 给它足够的时间
- 定期复盘
- 从中学习"
```

## 对话后行动

### 1. 记录决策日志

```
@jack-knowledge/decision-making/frameworks/decision-journal.md

完整记录整个探索和决策过程
```

### 2. 制定行动计划

具体的第一步、第二步、第三步

### 3. 设置复盘提醒

在日历上设置：
- 1周后
- 1个月后
- 3个月后

### 4. 开始行动

不要拖延，开始第一步

## 注意事项

1. **不要强求确定性**
   - 100%的确定是不可能的
   - 80%的清晰就可以行动
   - 剩下的20%在行动中澄清

2. **相信你的决策**
   - 你已经充分探索了
   - 你倾听了内心
   - 你做出了你的选择
   - 相信它

3. **接受不完美**
   - 没有完美的决策
   - 每个选择都有代价
   - 重要的是你做了选择

4. **准备好调整**
   - 决策不是终点
   - 可以根据情况调整
   - 保持觉察和学习
