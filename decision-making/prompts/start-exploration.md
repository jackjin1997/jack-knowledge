# 开始探索 Prompt

快速启动决策探索对话。

## 用途

当你面临决策时，用这个 Prompt 快速开始与决策教练的对话。

## 使用场景

- 刚开始思考一个决策
- 不知道从哪里开始
- 需要快速启动探索

## Prompt 模板

```
@jack-knowledge/decision-making/coaches/self-discovery-coach.md
@jack-knowledge/decision-making/methods/socratic-questioning.md

请作为我的决策教练（不是顾问），帮我探索以下决策：

## 决策主题
{简短描述你面临的决策}

## 当前状态
{你现在的情况}

## 我的困惑
{什么让你不确定、犹豫}

请记住：
1. 只通过提问引导我思考
2. 不要给我建议或答案
3. 帮我发现我内心真实的想法
4. 保持中立，不评判

我准备好开始探索了。
```

## 使用示例

### 示例1：职业决策

**输入**：
```
@jack-knowledge/decision-making/coaches/self-discovery-coach.md
@jack-knowledge/decision-making/methods/socratic-questioning.md

请作为我的决策教练，帮我探索以下决策：

## 决策主题
是否接受新工作 offer

## 当前状态
在现公司工作3年，稳定但感觉没有成长
新公司给了offer，薪资高20%，但是创业公司

## 我的困惑
不知道稳定重要还是成长重要
担心创业公司不稳定
但又觉得现在不挑战以后会后悔

请通过提问帮我探索内心真实的想法。
```

**预期对话**：
```
教练："当你想到'没有成长'时，你的感觉是什么？"

你："有点沮丧，感觉在浪费时间。"

教练："浪费时间...这触动了你什么？"

你："我担心35岁后竞争力下降..."

（通过问题层层深入）
```

### 示例2：生活选择

**输入**：
```
@jack-knowledge/decision-making/coaches/values-clarifier.md

请帮我探索决策：

## 决策主题
周末是陪家人还是加班完成项目

## 当前状态
项目deadline紧张，但已经连续几周没好好陪家人了

## 我的困惑
两个都重要，不知道如何选择
感觉选哪个都会愧疚

请帮我澄清价值观，发现我真正在意什么。
```

## 变体

### 变体1：紧急决策

```
我需要快速探索一个决策：

{决策描述}

我只有{时间}的时间。
请用最核心的问题帮我快速澄清。
```

### 变体2：复杂决策

```
我面临一个复杂的决策：

{详细背景}

涉及多个维度：
- 维度1
- 维度2
- 维度3

请系统地帮我探索各个维度。
```

### 变体3：重复决策

```
我又面临类似的决策了：

上次决策：{简述}
这次决策：{简述}

我想看看这次能有什么不同的发现。
```

## 准备工作

在开始探索前：

1. **找个安静的时间和空间**
   - 不被打扰
   - 可以专注思考

2. **准备好开放的心态**
   - 愿意诚实面对自己
   - 接受不确定性
   - 相信过程

3. **准备记录**
   - 可以边对话边记录
   - 或对话后整理

4. **没有时间压力**
   - 给自己足够的探索时间
   - 不要在deadline前最后一刻

## 对话提示

### 保持开放

- 不要预设答案
- 让探索自然展开
- 跟随教练的引导

### 诚实回答

- 说出真实的想法
- 不用"应该"的答案
- 承认"不知道"

### 注意感受

- 注意情绪变化
- 注意身体反应
- 直觉和感觉很重要

### 给予时间

- 不要急于回答
- 允许沉默和思考
- 深层答案需要时间浮现

## 何时停止

当你：
- ✅ 感到清晰了
- ✅ 知道自己真正想要什么
- ✅ 准备好做决策
- ✅ 或者需要时间消化

可以说：
```
"我想我已经有了一些清晰。
请帮我总结一下今天的探索，
然后我需要一些时间让它沉淀。"
```

## 对话后

### 1. 记录
```
"请帮我用决策日志格式记录今天的探索：
@jack-knowledge/decision-making/frameworks/decision-journal.md"
```

### 2. 给自己时间
- 让洞察沉淀
- 不要急于决策
- 必要时睡一觉

### 3. 继续探索（如需要）
```
"我想继续探索X方面：
@jack-knowledge/decision-making/coaches/[another-coach].md"
```

## 注意事项

1. **这不是一次就够的**
   - 重要决策可能需要多次探索
   - 可以用不同的教练
   - 可以用不同的方法

2. **答案会自己出现**
   - 不要强求
   - 相信过程
   - 真实的答案会浮现

3. **没有时间限制**
   - 给自己充分的时间
   - 宁可多探索，不要匆忙
