# 深入挖掘 Prompt

对某个方面进行深度探索。

## 用途

当你已经做了初步探索，但某个方面需要更深入挖掘时使用。

## 使用场景

- 发现了关键问题，需要深入
- 某个感受或想法需要理解
- 价值观需要澄清
- 某个恐惧需要探索

## Prompt 模板

```
@jack-knowledge/decision-making/coaches/{选择合适的coach}.md
@jack-knowledge/decision-making/methods/{选择合适的方法}.md

我想深入探索决策的某个方面：

## 决策背景
{简述决策}

## 已有的探索
{之前发现了什么}

## 需要深入的方面
{具体想探索什么}

例如：
- 某个特定的恐惧
- 某个价值观冲突
- 某个内心矛盾
- 某个模式
- 某个感受

请用{方法名}帮我深入探索这个方面。
```

## 深入探索的方向

### 1. 探索感受

```
"我想深入探索某个感受：

感受：{描述感受}

问题：
- 这个感受从何而来？
- 它在告诉我什么？
- 它背后是什么？"
```

### 2. 探索恐惧

```
"我想深入探索某个恐惧：

恐惧：{描述恐惧}

问题：
- 这个恐惧的根源是什么？
- 它是真实威胁还是想象？
- 最坏情况我能应对吗？"
```

### 3. 探索价值观

```
"我想深入理解某个价值观：

价值观：{如'自由'}

问题：
- 这对我意味着什么？
- 为什么这对我重要？
- 它从何而来？"
```

### 4. 探索模式

```
"我发现了一个重复的模式：

模式：{描述模式}

问题：
- 为什么我重复这个模式？
- 这个模式对我有什么功能？
- 它保护了什么？代价是什么？"
```

### 5. 探索矛盾

```
"我内心有个矛盾：

一方面：{想法/感受A}
另一方面：{想法/感受B}

问题：
- 这个矛盾说明什么？
- 两者能共存吗？
- 真正的冲突在哪里？"
```

## 深入探索技巧

### 1. 使用合适的方法

- **恐惧/动机** → 五个为什么
- **价值观** → 价值观对齐法
- **多面性** → 多角度探索
- **时间维度** → 未来自我对话

### 2. 给予足够深度

不要停在表面：
```
第1层："我担心"
第2层："为什么担心？"
第3层："担心背后是什么？"
第4层："这触动了我什么？"
第5层："根本原因是什么？"
```

### 3. 跟随能量

注意什么让你：
- 有强烈反应
- 停顿或沉默
- 情绪波动

这些指向重要的地方。

### 4. 允许意外

- 探索可能带来意外发现
- 不要固守预设方向
- 跟随探索的自然流动

## 何时需要深入探索

### 信号1：表面探索后仍不清晰

```
"我做了初步探索，但还是不清楚..."
→ 需要深入某个方面
```

### 信号2：发现了关键点

```
"我发现我真正担心的是X..."
→ 深入探索X
```

### 信号3：遇到矛盾或冲突

```
"我理性和感性的答案不一样..."
→ 探索这个矛盾
```

### 信号4：触碰到强烈情绪

```
"想到这个我就很焦虑/兴奋..."
→ 探索这个情绪
```

## 深入探索的层次

### 表层：事实和选项
"我面临A和B两个选择"

### 第2层：感受和担忧
"我担心X，我希望Y"

### 第3层：价值观和需求
"我真正在意的是Z"

### 第4层：根源和模式
"这和我从小的W经历有关"

### 核心层：本质和意义
"这个决策真正关于'我是谁'"

**深入探索的目标**：到达核心层。

## 探索深度检查

问自己：
```
- [ ] 我只是在分析选项，还是探索了内心？
- [ ] 我触碰到真实的感受了吗？
- [ ] 我找到价值观了吗？
- [ ] 我理解根源了吗？
- [ ] 我看到了本质意义吗？
```

如果前面的回答是"否"，继续深入。

## 对话后

### 如果更清晰了

```
"我想我清晰了。
请帮我用决策日志记录：
@jack-knowledge/decision-making/frameworks/decision-journal.md"
```

### 如果需要沉淀

```
"我需要一些时间消化今天的探索。
请帮我记录关键洞察，
我过几天再继续。"
```

### 如果想换个角度

```
"这个角度探索得差不多了。
我想从另一个角度看：
@jack-knowledge/decision-making/coaches/[another-coach].md"
```

## 注意事项

1. **不要过度分析**
   - 深入不等于过度分析
   - 注意感受，不只是思考
   - 如果陷入分析，回到感受

2. **保持觉察**
   - 探索过程中观察自己
   - 注意防御机制
   - 注意逃避

3. **信任过程**
   - 答案会自然浮现
   - 不要强求
   - 给予时间和空间
