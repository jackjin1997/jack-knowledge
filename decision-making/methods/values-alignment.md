# 价值观对齐法

检查决策是否与核心价值观一致。

## 方法概述

价值观对齐法帮助你检验决策选项与个人核心价值观的一致性程度。当决策与价值观一致时，你会感到内心平静；不一致时，会感到不安。

## 核心理念

- **价值观是指南针** - 指引人生方向
- **一致性带来平静** - 对齐时内心平和
- **冲突带来不安** - 不对齐时内心挣扎
- **觉察是关键** - 注意内心的感受

## 使用步骤

### 第1步：识别核心价值观

**问自己**：
```
1. "对我来说，人生中最重要的是什么？"
2. "我的3-5个核心价值观是什么？"
3. "我愿意为什么而奋斗？为什么而牺牲？"
```

**提示**：
如果不清楚，使用：
```
@jack-knowledge/decision-making/coaches/values-clarifier.md
```

**常见价值观**（仅供参考）：
- 成长、学习、挑战
- 家庭、关系、爱
- 自由、独立、选择
- 稳定、安全、确定性
- 创造、表达、贡献
- 成就、认可、影响力
- 健康、平衡、快乐
- 意义、目的、信仰

### 第2步：列出决策选项

**清晰列出**：
```
选项A：[具体描述]
选项B：[具体描述]
选项C：[具体描述]
```

### 第3步：逐一对齐检查

对每个选项，问：

**核心问题**：
```
1. "这个选项体现了我的哪些价值观？"
2. "这个选项违背了我的哪些价值观？"
3. "选择这个，我会离哪些价值观更近？"
4. "选择这个，我会离哪些价值观更远？"
```

**深入感受**：
```
1. "想到选择这个，我的感觉是什么？"
   - 平静、兴奋 → 可能对齐
   - 不安、抗拒 → 可能不对齐

2. "我的身体有什么反应？"
   - 放松、舒展 → 对齐
   - 紧张、收缩 → 不对齐

3. "我的直觉说什么？"
```

### 第4步：对齐度评分（可选）

为每个价值观打分（1-10）：

| 价值观 | 选项A | 选项B | 选项C |
|--------|-------|-------|-------|
| 成长   | 9     | 5     | 7     |
| 家庭   | 4     | 9     | 6     |
| 自由   | 7     | 6     | 8     |
| 稳定   | 3     | 8     | 6     |

**注意**：
- 数字是辅助，不是答案
- 重要的是打分过程中的思考
- 感受比数字更重要

### 第5步：整体感受检查

**核心问题**：
```
1. "整体来看，哪个选项最对齐我的价值观？"
2. "选择它时，我感觉'对'吗？"
3. "我的内心是平静还是不安？"
4. "5年后的我会认可这个选择吗？"
```

**矛盾检查**：
```
如果理性分析指向A，但感受指向B：
"这个矛盾告诉我什么？"
"哪个价值观我之前忽略了？"
"我的价值观排序是否需要重新审视？"
```

### 第6步：测试决策

**想象测试**：
```
1. "想象我选择了A，醒来后的感觉是...？"
2. "想象我选择了B，醒来后的感觉是...？"
3. "想象半年后，我会对哪个选择更满意？"
```

**承诺测试**：
```
1. "我准备好承诺选项A了吗？"
2. "说出'我决定选A'时，我的感觉如何？"
3. "有什么让我迟疑的吗？"
```

## 处理价值观冲突

### 情况：多个价值观相互冲突

**探索问题**：
```
1. "我重视X，也重视Y，但在这个决策中它们冲突了"
2. "在这个情境下，哪个价值观对我更核心？"
3. "我愿意在哪个价值观上妥协？不愿意在哪个上妥协？"
4. "有没有创造性的方式部分满足两者？"
```

**优先级排序**：
```
"如果只能选一个价值观来指导这个决策，
我会选...，因为..."
```

**接纳冲突**：
```
"也许这个冲突本身就是答案的一部分。
真实的生活就是在不完美中选择。
我能接纳这个不完美吗？"
```

## 价值观对齐的信号

### 高度对齐的信号

- ✅ 内心感到平静
- ✅ 感觉"对"、"顺"
- ✅ 有清晰感和确定感
- ✅ 身体放松
- ✅ 愿意承诺
- ✅ 说出决策时感到踏实

### 不对齐的信号

- ⚠️ 内心不安、纠结
- ⚠️ 感觉"别扭"、"不对劲"
- ⚠️ 需要说服自己
- ⚠️ 身体紧张
- ⚠️ 迟疑不决
- ⚠️ 说出决策时感到心虚

**如果发现不对齐**：
```
"这个不安在告诉我什么？"
"我的哪个价值观在抗议？"
"我真的要做这个决策吗？"
"还是我需要重新考虑？"
```

## 对齐检查清单

在做重要决策前，问自己：

**价值观层面**：
- [ ] 我清楚自己的核心价值观吗？
- [ ] 这个决策涉及哪些价值观？
- [ ] 各选项如何影响这些价值观？

**感受层面**：
- [ ] 想到选项A时，我的感觉是？
- [ ] 想到选项B时，我的感觉是？
- [ ] 哪个让我更平静/兴奋/踏实？

**未来层面**：
- [ ] 5年后的我会认可这个选择吗？
- [ ] 临终时回看，这个选择体现了什么？
- [ ] 我想成为的人会做这个选择吗？

**整合层面**：
- [ ] 理性分析和内心感受一致吗？
- [ ] 有什么不安的感觉吗？
- [ ] 我准备好承诺这个选择了吗？

## 示例：完整对齐分析

### 决策：是否接受新工作offer

**核心价值观**：
1. 成长 (9/10 重要度)
2. 家庭 (8/10 重要度)
3. 自由 (7/10 重要度)

**选项分析**：

**选项A：留在现公司**
- 成长：3/10（学不到新东西）
- 家庭：9/10（工作生活平衡好）
- 自由：7/10（时间自由）
- 感受：安全但沮丧

**选项B：接受新offer**
- 成长：9/10（很多学习机会）
- 家庭：5/10（可能加班多）
- 自由：4/10（创业公司压力大）
- 感受：兴奋但担忧

**对齐分析**：
```
最重要的价值观（成长）：B完全对齐
第二重要（家庭）：A更对齐
第三重要（自由）：A略优

内心感受：
- 想到A：平静但失落
- 想到B：紧张但兴奋

测试："我决定选B" → 感觉对，虽然有担心
     "我决定选A" → 感觉安全但遗憾
```

**洞察**：
- 核心价值观（成长）更重要
- 家庭的担忧可以通过其他方式平衡
- 内心真正渴望的是挑战
