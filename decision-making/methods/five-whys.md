# 五个为什么

连续问"为什么"，找到真实动机和根本原因。

## 方法概述

五个为什么（5 Whys）是一种深度探究技术，通过连续问"为什么"（通常5次），从表面现象深入到根本原因或真实动机。

## 核心理念

- **表面之下有真相** - 第一个答案往往不是真正的原因
- **层层深入** - 每个"为什么"都是一层深入
- **发现根源** - 找到真正的动机或原因
- **自我觉察** - 过程本身就是自我发现

## 如何使用

### 基本流程

```
陈述 → 为什么？→ 原因1 → 为什么？→ 原因2 → 为什么？→ ... → 根本原因
```

### 示例1：探索工作决策

```
陈述："我想换工作"

为什么？→ "现在的工作不满意"
为什么不满意？→ "没有成长空间"
为什么需要成长空间？→ "我想学新东西"
为什么想学新东西？→ "我怕被淘汰"
为什么怕被淘汰？→ "我担心失去价值，不被需要"

根本原因：安全感和自我价值感
```

### 示例2：探索购买决策

```
陈述："我想买那个名牌包"

为什么？→ "很好看"
为什么在乎好看？→ "别人会觉得我有品味"
为什么在乎别人的看法？→ "我想被认可"
为什么需要被认可？→ "我想证明自己成功"
为什么需要证明？→ "我担心不被重视，不够好"

根本原因：自我价值需要外界验证
```

## 提问技巧

### 1. 不要机械重复"为什么"

**变化问法**：
```
- "为什么这对你重要？"
- "这背后是什么？"
- "这意味着什么？"
- "你真正在意的是什么？"
- "这触动了你什么？"
- "再深一层，是什么？"
```

### 2. 注意情绪变化

当用户：
- 停顿更长
- 声音变化
- 情绪波动
- 说"从没想过这个"

说明接近核心了。

### 3. 不要强行到5次

- 5次是参考，不是规则
- 有时3次就够
- 有时需要7-8次
- 当用户说"对，就是这个！"时停下

### 4. 尊重防御机制

如果用户：
- 回避某个问题
- 转移话题
- 情绪不适

不要强行深入，给予空间：
```
"我注意到这个问题让你不太舒服，
我们可以换个角度，或者暂停这个探索。
你想怎么做？"
```

## 使用场景

### 1. 探索真实动机

**适用**：
- "我想要X"
- "我应该Y"
- "我必须Z"

**目标**：
找到"想要"背后真正的需求。

### 2. 分析犹豫不决

**适用**：
- "我不知道该不该..."
- "我很纠结..."

**目标**：
找到犹豫背后的根本顾虑。

**示例**：
```
"我犹豫要不要接受这个offer"
为什么犹豫？→ "担心不适应"
为什么担心？→ "新环境不熟悉"
为什么不熟悉让你担心？→ "可能做不好"
为什么担心做不好？→ "会让人失望"
为什么害怕让人失望？→ "我担心不被认可，证明不了自己"

根本问题：自我认可不足
```

### 3. 理解价值观

**适用**：
- 识别核心价值观
- 理解价值观冲突

**示例**：
```
"我重视稳定"
为什么？→ "稳定让我安心"
为什么需要安心？→ "我不喜欢不确定性"
为什么不喜欢？→ "不确定让我焦虑"
为什么焦虑？→ "我需要掌控感"

核心价值观：掌控感/安全感
```

### 4. 发现思维限制

**适用**：
- "我不能..."
- "我必须..."

**目标**：
发现限制背后的假设和信念。

## 五个为什么的变体

### 变体1：五个"还有呢？"

```
"我担心失败"
还有呢？→ "担心浪费时间"
还有呢？→ "担心被嘲笑"
还有呢？→ "担心证明不了自己"
还有呢？→ "担心父母失望"

（发现多层次的担忧）
```

### 变体2：五个"然后呢？"

```
"我选择了A"
然后呢？→ "会得到X"
然后呢？→ "会带来Y"
然后呢？→ "会导致Z"
然后呢？→ "最终影响..."

（看到长期后果）
```

### 变体3：五个"那又如何？"

```
"这个决策可能失败"
那又如何？→ "会损失投入"
那又如何？→ "需要重新开始"
那又如何？→ "会延迟计划"
那又如何？→ "其实...也没那么严重"

（看清恐惧的本质）
```

## 注意事项

### 1. 不是审问

- 保持好奇，不是质疑
- 语气温和，不是咄咄逼人
- 目的是探索，不是挑战

### 2. 允许"不知道"

```
用户："我不知道为什么"
教练："没关系，我们可以换个角度探索。
或者给自己一些时间，答案会来的。"
```

### 3. 注意循环

如果回答开始循环重复，说明：
- 可能已经到底层
- 或者触碰到防御机制
- 可以换个方向探索

### 4. 结合其他方法

五个为什么可以和其他方法结合：
- 和价值观澄清结合
- 和视角探索结合
- 和感受探索结合

## 示例对话

### 完整示例：职业决策

```
用户：我在考虑读MBA。

教练：为什么想读MBA？

用户：对职业发展有帮助。

教练：为什么需要这个帮助？

用户：我想升职到管理层。

教练：为什么想升职？

用户：更高的收入和地位。

教练：收入和地位对你意味着什么？

用户：...证明我成功了吧。

教练：为什么需要证明成功？

用户：（沉默）...可能是因为我一直觉得自己不够好，
      想证明给父母看，证明当年他们的投入是值得的。

教练：这是你发现的根本动机。你如何看待这个发现？

用户：原来我一直在为了证明而奋斗...
      我需要想想，这是我真正想要的吗？
```

**洞察**：
- 表面：想读MBA
- 第1层：职业发展
- 第2层：升职
- 第3层：收入和地位
- 第4层：证明成功
- **根源**：自我价值感、父母期望

**价值**：
用户自己发现了决策背后的深层动机，可以重新审视是否要读MBA，或者找到其他满足根本需求的方式。
