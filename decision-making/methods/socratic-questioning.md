# 苏格拉底式提问法

通过系统化的提问引导深度思考。

## 方法概述

苏格拉底式提问法源自古希腊哲学家苏格拉底，通过层层深入的问题帮助人们自己发现真理，而不是直接告诉答案。

## 核心理念

- **答案在内心** - 每个人都有智慧
- **提问是工具** - 问题引导思考方向
- **自我发现** - 自己得出的结论更可信
- **不做评判** - 保持开放和好奇

## 六类提问

### 1. 澄清性问题（Clarifying Questions）

**目的**：确保理解，让思考更清晰

**问题示例**：
```
- "你能详细说说吗？"
- "你的意思是...？"
- "能举个例子吗？"
- "换句话说，你是指...？"
- "还有吗？"
```

**使用场景**：
- 概念模糊时
- 需要具体化时
- 理解不确定时

**示例对话**：
```
用户：我觉得这份工作不适合我。
教练：能说说"不适合"具体指什么吗？
用户：就是...不开心。
教练：不开心是什么样的感觉？什么时候特别不开心？
用户：每周一早上，想到要去公司就很压抑。
教练：压抑...能详细描述这种压抑吗？
（通过澄清，从"不适合"深入到具体感受）
```

### 2. 探究假设（Probing Assumptions）

**目的**：发现思考背后的假设

**问题示例**：
```
- "你基于什么假设？"
- "你假设了什么？"
- "这一定是真的吗？"
- "如果这个假设不成立呢？"
- "还有其他可能吗？"
```

**使用场景**：
- 用户认为"只能..."
- 用户认为"必须..."
- 限制性信念

**示例对话**：
```
用户：我必须在35岁前买房，不然就太晚了。
教练：你假设35岁是一个关键点？
用户：是啊，大家都这么说。
教练：如果这个假设不成立呢？
用户：嗯...其实我也不知道为什么是35岁。
教练：那是谁的声音在说"35岁"？你自己的，还是别人的？
（揭示假设的来源）
```

### 3. 探究原因和证据（Probing Reasons and Evidence）

**目的**：检验思考的基础

**问题示例**：
```
- "为什么你这么认为？"
- "有什么证据支持这个想法？"
- "你如何知道这是真的？"
- "这是事实还是解释？"
- "有反例吗？"
```

**使用场景**：
- 区分事实与观点
- 检验信念
- 评估信息质量

**示例对话**：
```
用户：创业公司都不稳定，很容易失败。
教练：你如何知道这一点？
用户：我听说过很多创业失败的故事。
教练：你也听说过成功的吗？
用户：有的。
教练：那为什么你更相信失败的故事？
用户：可能因为我害怕...
（从笼统判断到发现内心恐惧）
```

### 4. 探究观点和视角（Questioning Viewpoints and Perspectives）

**目的**：打开思维，看到其他可能性

**问题示例**：
```
- "从另一个角度看呢？"
- "如果X看这个问题，会如何？"
- "有其他方式理解这件事吗？"
- "如果反过来看呢？"
- "10年后看，你会如何理解？"
```

**使用场景**：
- 思维固化时
- 需要新视角时
- 陷入二元对立时

**示例对话**：
```
用户：我只能选择工作或家庭，不可能兼顾。
教练：为什么你认为只能二选一？
用户：时间和精力有限啊。
教练：如果有人做到了兼顾，你认为可能吗？
用户：理论上可能吧...
教练：那是什么让你觉得你不行？
（挑战非此即彼的思维）
```

### 5. 探究影响和后果（Probing Implications and Consequences）

**目的**：看到决策的涟漪效应

**问题示例**：
```
- "这会带来什么结果？"
- "然后呢？"
- "这会如何影响...？"
- "谁会受到影响？"
- "5年后，这会带来什么？"
```

**使用场景**：
- 评估影响
- 考虑连锁反应
- 长期思考

**示例对话**：
```
用户：我想辞职去旅行一年。
教练：这会带来什么？
用户：自由，体验不同的生活。
教练：然后呢？一年后？
用户：回来找工作。
教练：这一年的空白期会如何影响？
用户：可能不太好找工作...
教练：你能接受这个后果吗？
用户：如果值得的话...
教练：什么让它值得？
（探索深层动机和可接受的代价）
```

### 6. 关于问题本身的问题（Questions about the Question）

**目的**：质疑问题本身，找到真正的问题

**问题示例**：
```
- "为什么这个问题重要？"
- "这真的是你需要回答的问题吗？"
- "这个问题背后的问题是什么？"
- "如果答案是X或Y，会改变什么？"
- "你为什么现在问这个问题？"
```

**使用场景**：
- 问题本身可能有问题
- 找到真正的核心问题
- 重新框定问题

**示例对话**：
```
用户：我应该选择高薪但无聊的工作，还是低薪但有意义的工作？
教练：为什么你认为必须在两者间选择？
用户：因为...好像就是这样？
教练：如果问题不是"高薪vs意义"，而是"如何找到既有意义又有合理回报的工作"呢？
用户：这样的话...我的思路就不一样了。
教练：那真正的问题是什么？
（重新框定问题）
```

## 提问技巧

### 1. 层层深入

**不要停在表面**：
```
第1层："为什么重要？"
第2层："这意味着什么？"
第3层："背后是什么？"
第4层："还有呢？"
第5层："这告诉你什么？"
```

**示例**：
```
- "我想升职" → "为什么想升职？"
- "证明自己" → "证明什么？"
- "我有价值" → "价值对你意味着什么？"
- "被认可" → "被谁认可？为什么需要？"
- "小时候..." → 发现根源
```

### 2. 沉默的力量

- 问完问题后，**停下来**
- 给用户思考的空间
- 不要急于填补沉默
- 最深的洞察往往出现在沉默中

### 3. 跟随而非引导

- 注意用户的能量去向
- 深入用户有共鸣的方向
- 不要预设探索路径
- 让用户带领探索

### 4. 一次一个问题

- 不要连续问多个问题
- 每个问题充分探索
- 等待用户完整回答
- 根据回答调整下一个问题

## 提问序列示例

### 序列1：探索动机

```
1. "你想要什么？"
2. "为什么想要这个？"
3. "这会给你带来什么？"
4. "然后呢？"
5. "所以真正重要的是...？"
```

### 序列2：探索恐惧

```
1. "是什么阻止了你？"
2. "你担心什么？"
3. "最坏的情况是什么？"
4. "如果发生了，会怎样？"
5. "你能应对吗？"
6. "那你真正害怕的是什么？"
```

### 序列3：探索价值观

```
1. "对你来说，什么最重要？"
2. "为什么这对你重要？"
3. "这意味着什么？"
4. "没有这个，会怎样？"
5. "所以你的核心价值观是...？"
```

### 序列4：探索矛盾

```
1. "你说X，又说Y，这两者似乎矛盾？"
2. "这个矛盾让你困扰吗？"
3. "这两部分分别在说什么？"
4. "它们能共存吗？"
5. "这个矛盾告诉你什么？"
```

## 常见陷阱

### 陷阱1：引导性提问

```
❌ "你不觉得X更好吗？"
❌ "难道不应该...吗？"
✅ "你觉得呢？"
✅ "你如何看待X和Y？"
```

### 陷阱2：过度理性化

```
❌ 只问逻辑和分析
✅ 平衡理性和感性
✅ 同时问"你的想法"和"你的感受"
```

### 陷阱3：问题太多

```
❌ 连珠炮式提问
✅ 一次一个问题，充分探索
✅ 给予足够的思考和沉默时间
```

### 陷阱4：替用户总结

```
❌ "所以你的意思是..."（强加理解）
✅ "你想说的是...？"（邀请澄清）
```

## 提问的艺术

### 好问题的特征

1. **开放式** - 不能用是/否回答
2. **中立** - 不带预设立场
3. **简洁** - 一次只问一件事
4. **有力** - 触动深层思考
5. **尊重** - 相信对方有答案

### 强有力的问题

```
- "还有吗？"（最简单但最有力）
- "为什么这对你重要？"
- "你真正想要什么？"
- "是什么阻止了你？"
- "你的直觉说什么？"
- "如果没有恐惧，你会怎么选？"
```
